📦 Deployment Options

🚀 Option 1: Streamlit Cloud (Recommended)

Deploy this app on Streamlit Cloud for free hosting with automatic updates.

1. Push this folder to GitHub
2. Go to https://share.streamlit.io
3. Click "New app"
4. Select your repository
5. Main file: app.py
6. Click "Deploy"
Deploy on Streamlit Cloud

🌐 Option 2: This Netlify Site

Current Site: https://pwd-tools-priyanka.netlify.app

Hindi Bill Note Sheet: https://hindibillnote.netlify.app

This landing page is hosted on Netlify. For the full Streamlit app, deploy to Streamlit Cloud (Option 1).

The Hindi Bill Note Sheet has been integrated into the main PWD Tools Suite with enhanced features.

🐳 Option 3: Docker (Self-Hosted)

Run the app in a Docker container on your own server.

docker-compose up -d

Access at: http://localhost:8501

💻 Option 4: Local Development

Run the app locally for development and testing.

pip install -r requirements.txt
streamlit run app.py
