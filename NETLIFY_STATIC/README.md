# Netlify Static Landing Page

This folder contains ONLY the static landing page for Netlify.

## Deploy to Netlify

1. Go to https://app.netlify.com
2. Login to your account
3. Find site: **pwd-tools-priyanka**
4. Drag and drop this **NETLIFY_STATIC** folder
5. Done!

## What This Contains

- `index.html` - Beautiful landing page
- `netlify.toml` - Netlify configuration

## For the Full Streamlit App

Deploy to **Streamlit Cloud** instead:
- See `DEPLOY_TO_STREAMLIT_CLOUD.md` in parent folder
- Streamlit Cloud is FREE and designed for Streamlit apps
- Netlify cannot run Streamlit apps (needs Python server)

## Your Sites

- **Netlify (Landing Page):** https://pwd-tools-priyanka.netlify.app
- **Streamlit Cloud (Full App):** Deploy following the guide above
