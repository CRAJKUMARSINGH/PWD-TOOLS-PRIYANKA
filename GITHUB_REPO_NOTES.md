# 🏗️ PWD Tools Suite v2.0

> **Professional Infrastructure Management Tools for Public Works Department**

## 🌟 About This Repository

This is a comprehensive suite of 13 professional tools designed specifically for the Public Works Department (PWD) to streamline infrastructure management, billing, financial calculations, and document generation.

## 🛠️ 13 Professional Tools Included

| Category | Tools |
|----------|-------|
| **📄 Document Generation** | Bill Generator, Bill Note Sheet, Hand Receipt (RPWA 28) |
| **💰 Financial Calculators** | EMD Refund, Security Refund, Deductions Table, Financial Progress |
| **📊 Batch Processing** | Excel to EMD (batch processing) |

## 🚀 Quick Start

### Option 1: Use Online (Easiest)
Just visit: https://pwd-tools-priyanka.streamlit.app

### Option 2: Run Locally
```bash
git clone https://github.com/CRAJKUMARSINGH/PWD-TOOLS-PRIYANKA.git
cd PWD-TOOLS-PRIYANKA
pip install -r requirements.txt
streamlit run app.py
```

### Option 3: Deploy Your Own (FREE)
1. Fork this repository
2. Go to https://share.streamlit.io
3. Connect GitHub and select this repo
4. Deploy! (Takes 2 minutes)
